# BotReminder MVP

Коротко о текущей версии:

- Telegram-бот для личных напоминаний и задач.
- Принимает текст и голосовые сообщения через OpenAI API.
- Понимает примерное время: "около 17", "после 14", "после обеда".
- Дожимает событие до реакции: "Вижу", "Выдвигаюсь", "Я тут", "Готово".
- Позволяет управлять событиями через `/list` и `/hot`.
- Умеет закрывать, удалять, переносить и откладывать события кнопками или текстом.
- Считает примерные расходы OpenAI API через `/cost`.
- Развернут на сервере как `systemd`-сервис `botreminder`.
